## Success metric

- LAND: GitHub stars, plugin installs, README clicks
- EXPAND: Cal bookings, demo calls, pilot conversions
- SPEAK: CFP accepted, talk delivered, leads from talk
- WRITE: Page views, signups attributed, replies/DMs from the piece
- PARTNER: Co-host confirmed, dinner shipped, portco intros
